# Simple Notepad

A simple text editor with a graphical user interface, written in C++ using the Qt framework. It allows opening, editing, and saving files in HTML format, as well as basic text formatting.

## Features

### File Operations
- Create a new document 
- Open an existing file 
- Save 
- Save As
- Check for unsaved changes when closing the window

### Editing
- Undo / Redo
- Cut, Copy, Paste

### Text Formatting
- **Font Style:** bold, italic, underline, strikeout
- **Font Family:** Sitka, Arial, Times New Roman, Impact, MS Serif
- **Font Size:** from 8 to 72 points
- **Text Color:** white, black, blue, red, green
- **Background Highlight:** semi-transparent cyan (#00f0f0 with alpha 120)
- **Alignment:** left, center, right

All formatting options are available both through the Format menu and the toolbar.

## Build Requirements
- Qt 5 or 6 (Core, GUI, Widgets modules)
- C++11-compatible compiler (e.g., GCC, MinGW, MSVC, Clang)
- CMake or qmake (depending on the build method)
